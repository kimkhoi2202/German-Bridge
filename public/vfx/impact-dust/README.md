# Impact Dust Textures

These four smoke/vapor particle textures are vendored for the German Bridge high-card landing impact effect.

- Source: OpenGameArt, "Smoke Vapor Particles"
- Author: Fupi
- License: CC0
- Source URL: https://opengameart.org/content/smoke-vapor-particles

Files:

- `smoke-vapor-1.png` from `smoke1.png`
- `smoke-vapor-2.png` from `smoke2.png`
- `smoke-vapor-3.png` from `smoke3.png`
- `smoke-vapor-4.png` from `smoke4.png`

The CSS uses these PNGs as alpha masks so the dust can inherit the local card-impact color while keeping the natural particle texture.
